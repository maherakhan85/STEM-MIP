# Images

Place your hero background image here.

**Hero section background:**  
- **Filename:** `hero-bg.jpg`  
- **Path used in site:** `/images/hero-bg.jpg`  
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

To use a different name or format, update the `background-image` in `src/app/globals.css` (search for `hero-bg.jpg`).

Recommended: at least 1920×1080 px for best quality on large screens.
